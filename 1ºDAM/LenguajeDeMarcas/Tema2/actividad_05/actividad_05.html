Actividad 5: Enlaces

Responde a las siguientes cuestiones en la página web para esta actividad:

  • ¿Qué es una página local?

    • Una pagina local es una pagina que se guarda y se ejecuta en tu propio dispositivo, sin necesidad de usar internet.

  • ¿Qué es una página remota?

    • Una pagina remota es una pagina que se guarda en algun servidor de internet, y se puede acceder a ella mediante internet y desde cualquier otro dispositivo.

  • ¿Qué se indica en href?

    • El href se usa para indicar a donde lleva el enlace.

  • ¿Qué ocurre si no se cierra la etiqueta A?

    • El navegador puede interpretar mal el enlace, afectando a mas contenido del que deberia , basicamente pueden ocurrir errores.
